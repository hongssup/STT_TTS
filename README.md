# SpeechToText_TextToSpeech iOS Demo
>Apple Naver Google STT &amp; TTS 비교 테스트


## Requirements
- iOS 13.0+
- Swift 5

## Installation
### Cocoapods
```
pod 'googleapis', :path => '.'
pod 'Alamofire'
```
